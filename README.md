### Objective:
Develop a classification model using the K-Nearest Neighbors (KNN) algorithm to predict the species of iris flowers based on the Iris dataset. Your task is to build the complete machine learning pipeline—from data preprocessing and splitting to model training, evaluation, and discussion of improvements.

### Data Preprocessing:

Instruction: Load the Iris dataset, describe its features, and check for missing values. Apply feature scaling (normalization/standardization) and encode any categorical variables.

### Data Splitting:

Instruction: Split the dataset into training, validation, and test sets using stratified sampling (e.g., 70% train, 15% validation, 15% test).

### Model Training & Hyperparameter Tuning:

Instruction: Train a KNN classifier (starting with k=3) on the training set. Tune hyperparameters (k value (3,5,7), distance metric(Euclidean, Manhattan)) using the validation set to optimize performance.

### Model Evaluation:

Instruction: Evaluate the model on the test set using accuracy, precision, recall, F1-score, and ROC-AUC. Additionally, implement leave-one-out cross-validation to further assess model robustness.
